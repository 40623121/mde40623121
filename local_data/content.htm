自動設定proxy

建立一個.reg檔  內容如下 :

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings]
"MigrateProxy"=dword:00000001
"ProxyEnable"=dword:00000001
"ProxyHttp1.1"=dword:00000000
"ProxyServer"="http://140.130.17.53:3128""ProxyOverride"="192.168.1.*;localhost"

4月份內容

CMSimfly網際內容管理系統

這是位於 heroku的網站，由於能在網頁中維護的 heroku網站必須與其他雲端儲存系統結合，而本網站是在近端維護後，提交推送到 heroku，並且可用來呈現網際內容管理相關程式的執行結果

如何使用此系統 ? 

創立一個heroku帳號和雲端倉庫 ( 需確認唯一性 ) 

在需要連結的資料夾內建立  .git目錄>git init

放入Profile>主程式內容. runtime.txt>告訴python的版本. requirements.txt>執行應用程式所需模組

連結>git remote add github 對應倉儲網址

如何傳送檔案至自己的CMSimfly系統 ? 

heroku login>git add . >git commit -am " ">git push heroku master

如何用瀏覽器看自己的CMSimfly系統 ? 

https://xxx.herokuapp.com

如何開啟近端CMSimfly系統 ? 

git clone>python  app_run.py >瀏覽器打開 http://localhost:5000

平行批次檔指令

由於在近端啟動 CMSimfly 網站時, 希望以 batch 指令同時啟動 Flask 網際程式與近端網頁, 這時需要利用能平行開啟執行的 MParallel 工具, 從 https://github.com/lordmulder/MParallel  下載 mparallet.exe 後, 放入可攜系統的 y:\ 目錄, 建立 run.bat 如下:

MParallel.exe --count=2 python app_run.py : C:\Program Files (x86)\Google\Chrome\Application\chrome.exe "http://localhost:5000"

啟動後便可同時執行近端 app_run.py 與 chrome 網頁開啟.

在近端維護網站內容

CMSimfly  在區網或廣域網路中執行, 登入後可以維護網站內容

Filezilla

Filezilla是一個FTP檔案傳輸軟體，可以一次將多個資料夾傳輸到遠方主機，Ex : ftp.nfu.edu.tw

使用Clonezilla單機複製與Clonezilla Server Edition的網路複製

Clonezilla是一個系統備份與還原的軟體，支援UEFI BIOS主機板GPT硬碟的複製、備份與還原。此軟體是使用1:1的方式來複製硬碟，也就是木的硬碟要大於等於來源硬碟。

複製步驟

>使用預設值開機

>選擇繁體中文

>不修改鍵盤對應

>進行再生龍

>選擇處理整顆硬碟或部分分割區的直接複製

>使用預設的選擇

>選擇複製本機硬碟至本機硬碟，並將硬碟連接在USB3.0的插槽，速度較快

>選擇來源硬碟sda，a代表第一顆硬碟，b代表第二顆硬碟

>使用第二顆硬碟當目的硬碟

>略去檢查與修正來源分割區的檔案系統

>輸入y

>再輸入y，確認執行

>輸入y，複製開機載入程序

接下來會出現以下畫面

>複製完成後，按Enter鍵

>選擇關機，結束複製

 

 

組員上課心得

蔡朝旭

從開課到現在已經教了很多東西，雖然一開始也是各種問號，想問組員意見結果都倒一片，從一開始的用cmd查詢自己的IP位址、一些關於網路的知識、使用VirtualBox虛擬主機與個人電腦連線，自動設定proxy，這些學問讓我省了很多步驟也達到網路就是再帶給人方便性的意義 ，還有利用Heroku架設個人網際內容管理系統，現在也開始教我們做網路線。網路的知識比我想像中寬廣，觸及和可用的地方也很多，雖然上學期比較好玩畢竟是網誌，希望這學期學完我也能當個更有內涵的人。

洪明琪

網際網路的課程這次跟以前用的東西不一樣，所以很多東西都是必須從頭來過，我們這組在建立Heroku雲端主機雖靠組長才得以建立起來，但是也從組長那學到一些建立的方法。網際網路對於設計的重要性透過老師在上課的提醒，越來越能感覺到對我們有很大的幫助，期望這學期能從老師身上學到更多對自己有用的工具。

潘季宏

這學期學了網際網路應用，學會了如何製作網頁及編輯程式，及網路的應用，讓我學習到了許多對電腦更進一步的認識，在途中，有幾個設定非常的不解，但是還是努力地去搞懂他，這堂課真的讓我學習到以前從不了解電腦的另一面。
